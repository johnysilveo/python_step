What is HTML?

Hypertext Markup Language (HTML) — стандартна мова розмітки для створення веб-сторінок і веб-застосунків. Браузери отримують HTML-документи з веб-сервера або локального сховища і відтворюють їх як сторінки.

HTML описує структуру сторінки семантично. Елементи позначаються тегами, записаними у кутових дужках, наприклад <p> або порожній тег <img>. Теги можуть мати атрибути, які додають додаткову інформацію.

Markup

Пара тегів складається зі start tag (наприклад, <h1>) та end tag (</h1>). Деякі елементи самозакривні — наприклад <img>.

Класичний приклад “Hello, World!”:

<!DOCTYPE html>
<html>
  <head>
    <title>This is a title</title>
  </head>
  <body>
    <p>Hello world!</p>
  </body>
</html>
